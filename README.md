## ğŸ“¦ Installation

1. **Option A: Direct Download**
   - Download `PsychedelicNexus.theme.css`
   - Place it in your BetterDiscord themes folder:
     - **Windows**: `%appdata%\BetterDiscord\themes`
     - **macOS**: `~/Library/Application Support/BetterDiscord/themes`
     - **Linux**: `~/.config/BetterDiscord/themes`
   - Enable the theme in BetterDiscord settings
   - **Note**: Background images will use GitHub raw URLs automatically

2. **Option B: Local Development**
   - Clone/download the entire repository
   - Place the theme file in your themes folder
   - Update line 26 in `PsychedelicNexus.theme.css` to use local path:
     ```css
     --nexus-bg-image: url("./assets/background-main.jpg");
     ```
   - Or use absolute path to assets folder
