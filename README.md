# Psychedelic Nexus Theme

## GitHub Repository

**Repository URL:** https://github.com/tshives26/discord-psychedelic-nexus

**Author:** SummitByte  
**Version:** 2.2.0

## Description

A sharp, modern, premium psychedelic theme with vibrant colors, trippy effects, and fully customized UI elements. Built on ClearVision framework.

## Installation

1. Download the `PsychedelicNexus.theme.css` file from the repository
2. Place it in your BetterDiscord themes folder
3. Enable it in BetterDiscord settings

## Auto-Update

This theme uses GitHub imports for automatic updates. When you reload Discord or the theme, it will pull the latest version from GitHub.

## Framework Files

- `main.css` - Main theme framework
- `betterdiscord.css` - BetterDiscord-specific styles

## Assets

- Background images and animated GIFs are stored in the `assets/` folder
